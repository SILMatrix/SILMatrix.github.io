Software Implementation Lifecycle Matrix (SILMatrix)

  • SILMatrix emerged as logic instrument for Software Implementation Project Partners to coordinate their activities in the process of software implementation. SILMatrix is elegantly intuitive both for experts and beginners.
  • SILMatrix systematizes concepts of Software Implementation Lifecycle by logical steps, organizational levels and implementation stages.
  • The origin of SILMatrix is zero point (Cell 00).
  • Column 1 bottomup demonstrates hierarchical organizational levels of Software Implementation Lifecycle (Cells 00..90).
  • The left-right (horizontal) direction demonstrates logical steps of Software Implementation Lifecycle on each level (Cells 01..09, 11..19, 21..29 etc).
  • The diagonal direction demonstrates successive stages of Software Implementation Lifecycle (Cells 00..99).
  • SILMatrix consists of 99 cells. Every cell contains a concept and shall be drilled down to description of methods. (WIP).
  • Start SILMatrix at Cell 00. Make 10 steps (Cells 01..09) on Level 0 to get over to Level 1 etc. Implementation Stage is generated at the intersection of the column and the row of the same order number. E.g. Stage 4 includes horizontal cells 40..49 and vertical cells 04..94.
  • The left-right (horizontal) direction, the bottomup (vertical) direction and diagonal direction are subcyclic.
  • Separable software solutions shall be at different Implementation Stages at a time.
